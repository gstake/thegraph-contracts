# Gstake

## Develop
### forge
https://book.getfoundry.sh/

### install
```
forge install 
```

### compile 
```
forge build
```

### test 
```
forge test --fork-url rpc...   --match-test test_some.. -vvvv
```

### verify
```
forge verify-contract --chain 42161  address .sol:contract --etherscan-api-key  your key
```
